### 关于美杜莎扫描器
	美杜莎扫描器是集漏洞扫描和端口扫描以及敏感信息收集等功能为一体的扫描器
	
### 使用说明

```bash
# 安装工具(ubuntu
apt-get install nmap
# 下载文件
git clone https://github.com/Ascotbe/Medusa.git
cd Medusa
# 安装依赖
pip install -r Medusa.txt
# 使用扫描器
python MedusaScan.py -u 192.168.0.1
```
