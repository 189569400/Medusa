### GitHub监控详细

`/api/github_monitor/`GitHubCVE数查询接口

```
{
	"token": "xxx"
}
```

> 参数解释

- `token`登录后返回的**token**

> 返回状态码

- 200：返回查询到的数据
- 404：非法查询哦宝贝！
- 403：小宝贝这是非法查询哦(๑•̀ㅂ•́)و✧
- 500：请使用Post请求
